Convert to ISMRMRD

The ismrmrd format is an open-source format for MR raw data. It contains meta information about acquisition parameters such as repetition time, echo times or excitation flip angle. In addition, the k-space dimensions and dimensions and resolution of the reconstructed images are provided. Each individual acquisition (i.e. each single readout/frequency encoding line also have a header. Additional information such as the k-space trajectory can also be added. Converters for vendor specific raw data formats to ismrmrd can be found here: ismrmrd@github

QC image reconstruction

In order to verify the uploaded raw data before it is added to the database, image reconstruction is carried out utilising SIRF. This is done as a quality control (QC) step. The reconstruction is run inside a docker container within the XNAT-based database. The container is open-source and can be build using this code: SIRF 4 QC
